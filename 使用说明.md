# AI电商海报生成系统 - 使用说明

---

## 📝 摘要版（100字）

AI电商海报生成系统是一款基于人工智能的自动化海报设计工具。用户只需上传产品图片并输入产品描述，系统即可通过AI自动分析产品特点、生成设计方案，并批量输出10张不同风格的专业海报。整个过程完全自动化，3-10分钟即可完成，大幅提升电商视觉设计效率。

---

## 📋 详情版（400字）

### 系统简介

AI电商海报生成系统采用先进的生成式AI技术，为电商运营者提供快速、专业的海报批量生成服务。系统由设计AI和图像生成AI协同工作，能够深度理解产品特性，自动创作符合品牌调性的视觉内容。

### 核心功能

**1. 智能设计方案生成**
- 上传1-9张产品图片
- 输入产品描述（包括卖点、工艺、目标人群等）
- AI自动分析并生成完整的视觉设计方案

**2. 多场景海报输出**
系统自动生成10张不同用途的海报：
- 主KV视觉海报
- 生活场景应用
- 产品细节展示
- 工艺卖点呈现
- 情感价值传递
- 使用场景演绎等

**3. 专业品质保证**
- 9:16竖版格式，适配电商平台
- 专业排版设计
- 品牌视觉统一
- 高清图片输出

### 技术特点

- **全自动流程**：从上传到生成，无需人工干预
- **并发生成**：同时处理5张海报，提升效率
- **实时进度**：可视化展示生成进度和状态
- **版本管理**：支持海报编辑和版本历史追溯
- **批量下载**：一键下载全部生成海报

### 适用场景

适合电商运营、品牌营销、产品推广等需要快速产出大量视觉物料的场景，特别适合新品上市、促销活动、多平台铺货等批量需求。

## 🎯 体验说明（800字）

### 一、使用流程

#### 第一步：环境准备

**系统要求**
- Python 3.8+
- Docker（可选，推荐生产环境使用）
- 现代浏览器（Chrome、Firefox、Safari等）

**配置环境变量**

首次使用需要配置 `.env` 文件：

```bash
# API配置
API_BASE_URL=https://ent.zetatechs.com/v1
API_KEY=your-api-key-here
DESIGN_MODEL_NAME=gemini-3-pro-preview-thinking
IMAGE_MODEL_NAME=gemini-3-pro-image-preview-flatfee
```

**启动方式**

*方式1：Docker部署（推荐）*
```bash
docker-compose up -d
# 访问 http://localhost:8003
```

*方式2：直接运行*
```bash
pip install -r requirements.txt
python server.py
# 访问 http://localhost:8000
```

#### 第二步：上传产品信息

**1. 填写产品描述**

在左侧文本框输入产品的详细信息，建议包含：
- **产品定位**：目标人群、使用场景、价值主张
- **核心卖点**：功能特点、技术优势、差异化价值
- **设计理念**：品牌故事、工艺细节、材质说明
- **情感诉求**：希望传递的情感和生活方式

示例：
```
这是一款手工制作的粗陶咖啡杯，不仅是饮品容器，更是清晨
冥想的仪式感载体。采用传统柴烧工艺，每一件都有独特的窑
变纹理。设计灵感源自日本侘寂美学，追求自然、简约、宁静
的生活态度。适合喜欢慢生活、注重品质的都市白领。
```

**2. 上传产品图片**

点击右侧上传区域，选择1-9张产品图片：
- **主图**：产品整体照片，展示完整外观
- **细节图**：纹理、工艺、材质特写
- **场景图**：使用场景、搭配效果
- **多角度图**：不同视角展示

图片要求：
- 格式：JPG、PNG、WEBP
- 建议尺寸：1080x1440以上
- 背景：建议干净简洁，突出产品主体

#### 第三步：开始生成

点击"开始创作"按钮后，系统进入自动生成流程：

**阶段1：上传素材（10%）**
将产品图片上传到服务器

**阶段2：AI分析产品（10-30%）**
设计AI深度分析产品描述和图片，理解产品特性、品牌调性、目标人群，生成完整的视觉设计方案

**阶段3：构思视觉语言（30-40%）**
解析设计方案，提取10个不同场景的海报主题和创作提示词

**阶段4：绘制海报作品（40-90%）**
图像生成AI并发创作10张海报，实时显示生成进度

**阶段5：完成（100%）**
所有海报生成完毕，自动跳转到成果展示页面

*预计耗时：3-10分钟（取决于网络和AI响应速度）*

#### 第四步：查看和下载

**成果展示**
- 10张海报以网格形式展示
- 每张海报包含编号、主题、视觉预览
- 点击海报可查看大图

**下载方式**
- **单张下载**：点击海报右上角下载按钮
- **批量下载**：点击"下载全部"按钮一次性获取所有海报

**海报编辑**（扩展功能）
- 全图修改：重新生成整张海报
- 局部修改：针对特定区域进行调整
- 版本管理：保留历史版本，随时回溯

### 二、注意事项

**1. 网络要求**
系统需要访问AI API服务，请确保网络畅通。如遇连接失败，请检查API配置和网络状态。

**2. 生成时间**
- 正常情况：3-5分钟
- 网络较慢：5-10分钟
- 如超过15分钟，建议刷新重试

**3. 浏览器缓存**
更新代码后如遇问题，请使用 `Ctrl+Shift+R`（Windows）或 `Cmd+Shift+R`（Mac）强制刷新浏览器。

**4. 磁盘空间**
每次生成约占用50-100MB空间，建议定期清理 `data/` 目录下的历史文件。

**5. API配额**
请注意AI服务的调用配额和费用，合理规划使用频率。

### 三、故障排查

**问题1：页面显示"离线"**
- 检查后端服务是否启动
- 验证端口8000/8003是否被占用
- 查看 `logs/api.log` 了解详情

**问题2：生成失败**
- 查看 `logs/task_*.log` 任务日志
- 检查 `.env` 文件中的API配置
- 验证API Key是否有效

**问题3：图片无法加载**
- 确认 `data/outputs/` 目录权限
- 检查浏览器控制台的网络请求
- 尝试直接访问图片URL

**问题4：生成的海报不符合预期**
- 优化产品描述，增加更多细节
- 提供更高质量的产品图片
- 明确品牌调性和视觉风格偏好

### 四、最佳实践

1. **产品描述要详细**：提供越多信息，AI生成的海报越精准
2. **图片质量要高**：清晰的产品图能显著提升生成效果
3. **描述品牌调性**：明确风格偏好（简约/复古/时尚等）
4. **突出核心卖点**：强调最重要的1-3个产品特点
5. **保存设计方案**：`data/designs/` 中的设计输出可复用参考
